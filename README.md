# Print-Matrix-in-snake-Pattern-
# Input:
2<br/>
3<br/>
45 48 54 21 89 87 70 78 15<br/>
2<br/>
25 27 23 21<br/>

# Output:
45 48 54 87 89 21 70 78 15<br/>
25 27 21 23<br/>

# Explanation:
# Testcase 1: Matrix is as below:
45 48 54<br/>
21 89 87<br/>
70 78 15<br/>

# Printing it in snake pattern will lead to the output as 45 48 54 87 89 21 70 78 15.
